Or Nominate via Pull Request

  1. Fork this repository
  2. Copy nominations/TEMPLATE.toml to nominations/{username}.toml
  3. Fill in their GitHub username and evidence
  4. Open a Pull Request

Our Dagger pipeline will automatically:

  • Fetch their last 90 days of GitHub activity
  • Clone their top repos and run desloppify
  • Calculate a Vibe Score based on commit velocity, LOC explosion, AI patterns, and code slop

Eligibility Rules

  • ≥100 commits in the last 90 days
  • Evidence of rapid-fire commit patterns
  • Significant repo proliferation
  • Desloppify score indicating high "slop" factor

Minimum Vibe Score to qualify: 60/100
